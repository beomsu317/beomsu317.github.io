Return-Oriented Programming - x64

Beomsu Lee Feb 3, 2020 2020-02-03T00:00:00+09:00
2 min

Description

x64의 경우 호출 규약 때문에 피연산자가 매우 중요하다.

x64의 ROP에서 POP 명령의 역할

ESP의 레지스터 값을 증가시켜 함수를 연속적으로 호출한다. 호출할 함수에 전달될 인자 값을 레지스터에 저장한다.

  • 호출할 함수의 1 번째 인자 값 저장: pop rdi; ret
  • 호출할 함수의 2 번째 인자 값 저장: pop rsi; ret
  • 호출할 함수의 1,3 번째 인자 값 저장: pop rdi; pop rdx; ret

// gcc rop.c -o rop -ldl -fno-stack-protector 
#define _GNU_SOURCE
#include <stdio.h>
#include <unistd.h>
#include <dlfcn.h>
 
void vuln(){
    char buf[50];
    void (*printf_addr)() = dlsym(RTLD_NEXT, "printf");
    printf("Printf() address : %p\n",printf_addr);
    read(0, buf, 256);
}
 
void main(){
    seteuid(getuid());
    write(1,"Hello ROP\n",10);
    vuln();
}
